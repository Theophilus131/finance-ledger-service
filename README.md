# finance-ledger-service





> A production-grade finance API built with Spring Boot 3, PostgreSQL, and JWT Security.



\##  In Progress



Full documentation coming after deployment.



\## Tech Stack

\- Java 17

\- Spring Boot 3

\- PostgreSQL 15

\- Flyway

\- Docker

\- JWT + HMAC Security

\- OpenAPI/Swagger

\- Thymeleaf PDF



\## Setup

Copy `application.yml.example` to `application.yml` and fill in your values.



Start the database:

\\```bash

docker-compose up -d

\\```



Run the app:

\\```bash

mvn spring-boot:run

\\```
